Launch Wait for Date

Launch Wait for Date: Use this option to set a date when you would like users to be able to enter the desired screen. You can configure the Alert Title and Alert Message presented if the date has not passed


Date - Must be formatted as 11-25-13: (Example: If 12-25-13 is entered, users will be allowed to use the screen on the 25th)

Alert Title: (Default: Check Back Later)

Alert Message: (Default: This screen will be availible soon.)
